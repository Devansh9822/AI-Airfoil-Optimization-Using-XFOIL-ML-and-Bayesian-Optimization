# Airfoil Optimization using XFOIL, Machine Learning and Bayesian Optimization
End-to-end aerodynamic optimization framework for NACA 4412 using CST parameterization, XFOIL simulations, machine learning surrogate models, and Bayesian optimization.

This project presents an framework for aerodynamic optimization of the NACA 4412 airfoil.

The workflow combines:

- CST parameterization
- XFOIL aerodynamic simulations
- Machine Learning surrogate models
- Bayesian Optimization

to identify airfoil geometries with improved aerodynamic efficiency.
